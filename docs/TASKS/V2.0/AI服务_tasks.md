# AI 服务 V2.0 开发任务清单 (AI服务_tasks.md)

> **最后更新**：2026-05-31 | **版本**：V2.0 | **整体状态**：🔴 待启动

## 模块技术上下文

- **微服务模块**：`lest-ai`（端口: 9405）
- **主目录**：`backend/lest-modules/lest-ai`
- **包路径**：`com.lest.modules.ai`
- **设计理念**：AI 不是独立功能模块，而是贯穿所有业务模块的**中央智能引擎**（AI-Native）
- **其他模块调用方式**：各业务模块通过 Feign API 调用 `lest-ai`，而非自行实现 AI 逻辑
- **版本说明**：V2.0 AI 服务在 [docs/PRD/V2.0/AI服务.md](../1-prd/V2.0/AI服务.md) 中有详细设计，本任务单对应 V2.0 版本的开发任务
