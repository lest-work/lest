# V5.0 小版本发布计划

> **发布时间线**：2028-03-17 ~ 2028-12-31（约 9 个月）
>
> **发布节奏**：每 1 周开发 + 0.5 周测试 = **每 1.5 周发布一个小版本**
>
> **版本命名规范**：`v5.{小版本}`，如 `v5.0.0-alpha`、`v5.0.1`
>
> **前置条件**：V4.0.6 正式发布已完成
>
> **核心目标**：面向超大型企业（金融、政务、教育等），高可用、高安全、高合规。提供开源基础版 + 商业企业版双版本。
