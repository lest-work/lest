# V2.0 小版本发布计划

> **更新时间**：2026-06-02
>
> **发布时间线**：2026-09-07 ~ 2026-11-28（约 3 个月）
>
> **发布节奏**：每 1 周开发 + 0.5 周测试 = **每 1.5 周发布一个小版本**
>
> **版本命名规范**：`v2.{小版本}`，如 `v2.0`、`v2.1`、`v2.4`
>
> **前置条件**：V1.5 正式发布（通知框架 + 插件系统框架 + 开放平台框架 + 主要插件）已完成

---

## V2.0 核心目标

在 V1.0 基础上，实现 **完整 Jira 方案**：EAV 自定义字段 + Screen/Scheme + 工作流引擎 + 项目模板，使每个团队都能定制最适合自己的工作方式。同时深化 UI 插件化（更多扩展点）和插件 SDK。

---
