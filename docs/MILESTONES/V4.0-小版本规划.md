# 4.0 小版本发布计划 — 插件生态与 AI

> **发布时间线**：2027-09-06 ~ 2027-12-10
>
> **路线原则**：V4.0 是插件正式开发的起点。V1.0/V2.0/V3.0 只做 Core 和扩展契约，不开发官方插件。
>
> **前置条件**：V3.0 Jira Core 完整版已发布，核心项目、Issue、Scheme、报表、自动化、搜索和导入导出稳定。

## 4.0 核心目标

V4.0 的目标是把 V1~V3 沉淀的扩展点变成真实插件平台，并交付第一批官方插件。AI 也作为可插拔能力接入，不进入 V1~V3 Core。

## 4.0 范围

| 模块 | 交付 |
|------|------|
| 插件运行时 | 安装、启用、禁用、配置、卸载、健康检查 |
| 插件 SDK | 后端 SDK、前端 SDK、事件订阅、API Client |
| 扩展点注册中心 | Notification、Workflow Action、Automation Action、Import/Export、Dashboard Widget、Task Detail Tab、Project Tab |
| 插件管理 UI | 插件列表、启停、配置、权限声明、审计 |
| 官方 IM 插件 | 飞书、钉钉、企业微信通知插件 MVP |
| 官方 DevOps 插件 | 代码仓库、CI/CD 插件 MVP |
| 官方协作插件 | 会议、WakaTime、报表插件 MVP |
| AI 插件 | AI Provider、AI 助手、AI 任务拆解、自然语言查询、风险识别、AI 报表 |

## 4.0 不做

| 功能 | 后续版本 |
|------|----------|
| SSO、LDAP、SCIM、企业组织同步 | V5.0 企业商业版 |
| 多租户、租户控制台、商业授权 | V5.0 企业商业版 |
| Portfolio、OKR、资源容量管理 | V5.0 企业商业版 |
| 企业级 SLA、高可用商业支持 | V5.0 企业商业版 |
