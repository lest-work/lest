# V3.0 小版本发布计划

> **更新时间**：2026-06-02
>
> **发布时间线**：2026-12-01 ~ 2027-04-03（约 4 个月）
>
> **发布节奏**：每 1 周开发 + 0.5 周测试 = **每 1.5 周发布一个小版本**
>
> **版本命名规范**：`v3.{小版本}`，如 `v3.0`、`v3.1`、`v3.5`
>
> **前置条件**：V2.5 正式发布（EAV + Screen/Scheme + 工作流引擎 + 项目模板，2026-11-28 完成）已开始，开发团队同步启动 V3.0
>
> **核心目标**：完成 Jira 功能 100% 对齐收口 + 中国企业协作增强（IM 深度集成 / 审批流 / Portfolio / OKR）
