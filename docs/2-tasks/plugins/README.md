## 插件开发原则

- **Core First**：先完成 Core，再并行开发插件
- **独立测试**：每个插件独立测试，不依赖其他插件
- **SDK 先行**：先完善插件 SDK，再开发官方插件
- **文档同步**：每个插件必须有完整的使用文档
