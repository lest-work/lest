# 报表中心 — V5.0 PRD

> **模块**: Core V5.0 | **许可**: 🔒 商业版

## 1. 产品概述

### 1.1 目标

提供企业级 BI 报表能力，支持多数据源接入、高级可视化、自定义报表和权限控制。

## 2. 核心功能

|| 功能 | 说明 | 优先级 |
||------|------|--------|
| 多数据源 | LEST 数据库 + 外部数据源（MySQL/PostgreSQL/CSV） | P0 |
| 拖拽式报表设计器 | 字段拖入画布自动生成图表 | P0 |
| 高级可视化 | 地图/漏斗/雷达图/桑基图 | P0 |
| 权限控制 | 报表分享 + 行级权限 | P0 |
| 订阅推送 | 定时邮件/IM 推送 | P1 |
| 报表嵌入 | IFrame 嵌入到外部系统 | P1 |

## 3. 报表类型

|| 类型 | 说明 |
||------|------|
| 柱状图/条形图 | 类别对比 |
| 折线图/面积图 | 趋势分析 |
| 饼图/环形图 | 占比分析 |
| 散点图 | 相关性分析 |
| 地图 | 地理分布 |
| 漏斗图 | 转化漏斗 |
| 雷达图 | 多维度对比 |
| 桑基图 | 流向分析 |
| 仪表盘 | KPI 仪表 |
| 表格 | 明细数据 |

## 4. 数据库设计

```sql
CREATE TABLE report (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    tenant_id      BIGINT NOT NULL,
    name            VARCHAR(128) NOT NULL,
    description     TEXT,
    chart_config    JSON NOT NULL COMMENT '图表配置（类型/字段/样式）',
    data_source     VARCHAR(32) DEFAULT 'lest',
    sql_query       TEXT COMMENT '自定义 SQL（数据源为 external 时）',
    share_type      VARCHAR(16) DEFAULT 'PRIVATE' COMMENT 'PRIVATE/PROJECT/PUBLIC',
    created_by      BIGINT NOT NULL,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at      DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

CREATE TABLE report_subscription (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    report_id       BIGINT NOT NULL,
    user_id         BIGINT NOT NULL,
    schedule        VARCHAR(32) NOT NULL COMMENT 'Cron 表达式',
    channels        JSON NOT NULL COMMENT '邮件/IM',
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
