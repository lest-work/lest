# 基础多租户 — V5.0 PRD

> **模块**: Core V5.0 | **许可**: 开源（MIT）

## 1. 产品概述

### 1.1 目标

为 SaaS 服务商或大型集团提供基础的多租户隔离能力。每个租户拥有独立的数据空间和配置。

### 1.2 隔离策略

|| 隔离维度 | 实现方式 |
||---------|---------|
| 数据隔离 | tenant_id 列级隔离 |
| 配置隔离 | 租户独立配置（Logo/域名/功能开关） |
| 缓存隔离 | Redis key 添加租户前缀 |

## 2. 核心功能

|| 功能 | 说明 | 优先级 |
||------|------|--------|
| 租户注册 | 创建租户 + 初始化 | P0 |
| 租户配置 | Logo/域名/品牌设置 | P0 |
| 租户路由 | 域名/Tenant-Id Header 切换租户 | P0 |
| 租户配额 | 资源限制检查 | P1 |
| 租户管理控制台 | 平台管理员查看所有租户 | P2 |

## 3. 数据库设计

```sql
CREATE TABLE tenant (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    name            VARCHAR(128) NOT NULL,
    domain          VARCHAR(255) UNIQUE,
    plan            VARCHAR(32) DEFAULT 'free',
    status          VARCHAR(16) DEFAULT 'active',
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at      DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

CREATE TABLE tenant_config (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    tenant_id       BIGINT NOT NULL,
    config_key      VARCHAR(64) NOT NULL,
    config_value   TEXT,
    UNIQUE(tenant_id, config_key)
);

-- 所有用户表新增 tenant_id 字段
ALTER TABLE user ADD COLUMN tenant_id BIGINT;
ALTER TABLE project ADD COLUMN tenant_id BIGINT;
ALTER TABLE issue ADD COLUMN tenant_id BIGINT;
```

## 4. 核心流程

1. **租户注册** → 创建 tenant 记录 + 初始化配置
2. **租户路由** → 解析域名/Header → 设置 ThreadLocal tenant_id
3. **数据隔离** → 所有查询自动注入 tenant_id 过滤条件（AOP）
4. **租户配置** → 读取 tenant_config 动态生效
