# 协作插件 PRD

> **版本归属**：V4.0.4
>
> **插件范围**：WakaTime、会议、报表、Jira Sync MVP。

## 产品目标

补齐 Core 之外的团队协作增强能力，让开发活动、会议行动项、跨项目摘要和 Jira 增量同步可以通过插件接入，而不污染 V1.0~V3.0 Jira Core 主链路。

## 功能范围

| 插件 | 能力 | 说明 |
|------|------|------|
| WakaTime | 用户绑定 | 用户绑定 WakaTime API Key 或 OAuth |
| WakaTime | 编码心跳 | 按项目/Issue Key 聚合编码时长 |
| 会议 | 会议记录 | 会议、参会人、纪要、行动项 |
| 会议 | 行动项转 Issue | 从会议行动项创建或关联 Issue |
| 报表 | 插件化 Gadget | 跨项目摘要、趋势卡片、可配置 Gadget |
| Jira Sync | 增量同步 | Jira Issue 增量同步，不替代 V3 导入导出 |

## 不做

| 能力 | 归属 |
|------|------|
| Core 工时模型 | V1.0 不做；如需工时由插件提供 |
| 企业绩效考核 | V5.0 企业报表/治理 |
| 双向复杂同步冲突策略 | 后续 Jira Sync 增强 |

## 验收标准

1. 任一协作插件关闭后，Core 项目、Issue、Board、Sprint 仍完整可用。
2. 会议行动项可以创建或关联 Issue，并保留来源信息。
3. 报表 Gadget 通过插件扩展点挂载，不硬编码进 Core 仪表盘。
4. Jira Sync 失败时可重试，且不会覆盖 Core 本地手动编辑。
