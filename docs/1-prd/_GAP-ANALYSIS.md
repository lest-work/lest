## 四、Jira vs LEST 故意不做的功能

以下 Jira 功能在 LEST 中**明确不做**，属于设计决策而非缺口：

| Jira 功能 | 原因 |
|---------|------|
| Confluence 集成（文档）| 独立产品，不做捆绑 |
| Jira Service Management（客服工单）| 不同产品线 |
| Bitbucket/GitHub 官方集成（代码托管）| 通过插件支持 |
| Trello 视图（看板简化版）| 已有自己的 Kanban |
| Jira移动端 App | V4.0+ 规划 |
| 项目类别管理 | 非核心需求 |
| 公告管理（项目级）| 用通知系统代替 |
