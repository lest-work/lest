# LEST Platform — 里程碑规划

> **更新时间**：2026-06-03
>
> **产品愿景**：LEST 是面向中国及全球企业的 Jira-like 敏捷管理平台。当前采用 **Core-first** 路线：先完成 Jira Core 的真实可用闭环，再开放插件生态、AI 插件能力和商业企业能力。
>
> **主路线图**：[ROADMAP-CORE-FIRST.zh-CN.md](./ROADMAP-CORE-FIRST.zh-CN.md)
>

## 路线原则

1. **V1.0 不开发官方插件**：会议、CI/CD、WakaTime、绩效、AI、IM 集成都不进入 V1.0 交付。
2. **先做可用 Core**：所有版本必须优先保证真实可运行、权限可验证、接口和数据一致。
3. **扩展矩阵先做契约**：V1.0/V2.0 只预留扩展点和事件命名，不承诺插件市场、热加载、完整 SDK。
4. **Scheme 是 V2.0 主线**：把 Jira 的可配置能力作为产品护城河。
5. **V3.0 收口 Jira Core**：补齐高频 Jira Core 能力，不把中国企业协作插件混入 Core 发布阻塞项。
6. **V4.0 才进入生态和 AI**：插件运行时、官方插件池和 AI 能力在 Core 稳定后启动。
7. **V5.0 明确商业边界**：多租户、SSO/LDAP、审计合规、高级报表、SLA 进入企业版。
