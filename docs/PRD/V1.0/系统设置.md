## 1. 概述

### 1.1 功能背景

系统设置是 LEST Platform 的基础设施模块，提供平台级的配置管理、日志审计、数据管理等功能，确保平台的可维护性和可扩展性。

### 1.2 功能目标

- 提供系统参数配置管理
- 提供操作日志和审计
- 提供数据备份与恢复
- 提供系统监控和状态查看
- 提供在线用户管理
- 提供系统公告管理
- 支持多租户配置（可选）

### 1.3 目标用户

- **超级管理员**：管理平台级配置、查看审计日志
- **租户管理员**：管理租户内配置
- **审计人员**：查看操作审计记录

## 2. 用户故事

- [ ] 支持分组查看参数（安全/通知/存储/集成）
- [ ] 支持修改参数值
- [ ] 支持参数变更记录

### 3.2 系统参数分类

```java
public enum SystemParamCategory {
    SECURITY("安全设置", Arrays.asList(
        "password.min_length",      // 密码最小长度
        "password.max_length",      // 密码最大长度
        "password.require_special", // 密码必须包含特殊字符
        "session.timeout",          // 会话超时时间(分钟)
        "login.max_failures",       // 登录最大失败次数
        "login.lock_duration",      // 登录锁定时长(分钟)
        "jwt.access_token_ttl",     // Access Token 有效期(秒)
        "jwt.refresh_token_ttl"     // Refresh Token 有效期(秒)
    )),

    NOTIFICATION("通知设置", Arrays.asList(
        "notification.email.enabled",
        "notification.dingtalk.enabled",
        "notification.wecom.enabled",
        "notification.feishu.enabled"
    )),

    STORAGE("存储设置", Arrays.asList(
        "storage.type",             // local / oss
        "storage.oss.endpoint",
        "storage.oss.bucket",
        "storage.max_file_size"     // 最大文件大小(MB)
    )),

    INTEGRATION("集成设置", Arrays.asList(
        "integration.github.enabled",
        "integration.gitlab.enabled",
        "integration.jenkins.enabled"
    ));
}

```

Authorization: Bearer {accessToken}

  "code": 200,
  "message": "success",
  "data": {

## 7. 错误码

### 7.1 系统模块错误码 (9500-9599)

> **注意**：系统设置模块使用 9500-9599 范围，保留 9500-9549 用于系统级通用错误码，9550-9599 用于系统设置特定错误码。9000-9499 由其他模块使用。

| 错误码 | 枚举常量 | HTTP 状态码 | 说明 |
|---------|----------|-------------|------|
| 9500 | `PERMISSION_DENIED` | 403 | 权限不足 |
| 9501 | `RESOURCE_NOT_FOUND` | 404 | 请求的资源不存在 |
| 9502 | `RESOURCE_CONFLICT` | 409 | 资源冲突 |
| 9503 | `INVALID_PARAMETER` | 400 | 参数校验失败 |
| 9504 | `SERVICE_UNAVAILABLE` | 503 | 服务暂不可用 |
| 9550 | `CONFIG_NOT_FOUND` | 404 | 配置项不存在 |
| 9551 | `CONFIG_UPDATE_FAILED` | 500 | 配置更新失败 |
| 9552 | `AUDIT_LOG_NOT_FOUND` | 404 | 审计日志不存在 |
| 9553 | `BACKUP_NOT_FOUND` | 404 | 备份记录不存在 |
| 9554 | `BACKUP_IN_PROGRESS` | 409 | 备份进行中 |
| 9555 | `RESTORE_IN_PROGRESS` | 409 | 恢复进行中 |
| 9556 | `RESTORE_FAILED` | 500 | 恢复失败 |
| 9557 | `SESSION_NOT_FOUND` | 404 | 会话不存在 |
| 9558 | `ANNOUNCEMENT_NOT_FOUND` | 404 | 公告不存在 |
