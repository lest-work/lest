# LEST Platform H5 移动端

## 项目介绍

LEST Platform 的 H5 移动端，为移动用户提供轻量级的访问体验。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite 6.x
- **UI 组件**: Vant 4.x (移动端组件库)
- **状态管理**: Pinia 3.x
- **路由**: Vue Router 4.x
- **HTTP**: Axios

## 功能模块

- [ ] 登录/注册
- [ ] 首页仪表盘
- [ ] 任务管理（移动端优化）
- [ ] 看板视图
- [ ] 通知消息
- [ ] 个人中心

## 开发指南

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```
